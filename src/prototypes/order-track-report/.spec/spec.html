订单跟踪报表

本规格为当前原型的事实依据。本轮按用户提供的 1917×923 登录后截图，完整还原 IOC 页面外壳、左侧静态菜单与订单跟踪报表；源页面视觉优先于通用主题。

当前方案

  • 使用场景：业务人员按订单、客户、渠道、产品与生产状态检索订单进度，并横向查看完整跟踪字段。
  • 页面范围：完整桌面端 IOC 运营平台页面，包含顶部品牌栏、面包屑、用户区、页签栏、左侧菜单、筛选区、查询/导出、宽表、滚动条、分页和水印。
  • 数据策略：使用与截图形态一致的脱敏模拟数据，不连接真实接口，不产生真实导出文件。
